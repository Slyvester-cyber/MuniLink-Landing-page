# MuniLink landing page

A fast, accessible landing page for MuniLink, an AI consulting and audit
venture serving South African local government.

## Run locally

```bash
npm start
```

Open <http://localhost:4173>.

## Validate

```bash
npm test
```

The site has no runtime dependencies or build step. The hero uses a small raw
WebGL shader with a CSS fallback and automatically becomes static when reduced
motion is preferred.
